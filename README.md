## üöÄ Setup Instructions

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, or Edge)
- No server or build tools required
- Internet connection (for CDN resources)

### Installation Steps

1. **Download or Clone the Project**
   - Simply download the project zip file and extract

2. **Open the Project**
   - Simply open `index.html` in your web browser
   - Or use a local development server for better performance:
     ```bash
     # Using Python 3
     python -m http.server 8000
     
     # Using Node.js (if you have http-server installed)
     npx http-server
     ```

3. **Access the Website**
   - If opened directly: `file:///path/to/CivicGrid/index.html`
   - If using a local server: `http://localhost:8000`

### Customization

#### Colors
Modify the CSS variables in `styles.css`:
```css
:root {
    --municipal-blue: #005B96;
    --efficient-green: #43A047;
    --alert-orange: #FB8C00;
}
```

#### Content
- Update statistics and metrics in `index.html`
- Modify chart data in `script.js` (Chart.js configurations)
- Customize platform modules and features in the HTML sections

#### Charts
Edit the chart data in `script.js`:
- Traffic chart data (lines 19-38)
- Impact metrics chart data (lines 97-112)
